Getting Started

1 Setup Authentication

  1. Go to Seeq → Profile → Access Keys → Create
  2. Copy both the Access Key and Password
  3. Enter your Seeq server URL, Key, and Password above
  4. Click "Authenticate"

2 Excel Functions

PULL Get time series data
=SqExcel.PULL(A1:C1,"2024-01-01T00:00:00","2024-01-02T00:00:00") =SqExcel.PULL(A1:C1,"8/1/2025 0:00","8/2/2025 0:00","points",500) =SqExcel.PULL(A1:C1,"8/1/2025 0:00","8/2/2025 0:00","grid","15min")

sensorNames: Range like A1:C1 with sensor names

startDatetime: "2024-01-01T00:00:00" or "8/1/2025 0:00"

endDatetime: "2024-01-02T00:00:00" or "8/2/2025 0:00"

mode: "points" (default) or "grid"

modeValue: Number of points (1000) or interval ("15min", "1h", "1d")

CURRENT Get current sensor value
=SqExcel.CURRENT("Area/Temperature")

sensorName: Single sensor name as text

Returns: Current scalar value

AVERAGE Calculate average over time range
=SqExcel.AVERAGE("Area/Temperature","8/1/2025 0:00","8/1/2025 1:00")

sensorName: Single sensor name as text

startDatetime, endDatetime: Time range for averaging

Returns: Average value over 100 data points

CREATE_PLOT_CODE Generate Python plotting code (supports multiple sensors)
=SqExcel.CREATE_PLOT_CODE(A1:C1,"8/1/2025 0:00","8/1/2025 1:00") =SqExcel.CREATE_PLOT_CODE(A1:B1,"8/1/2025","8/2/2025",100,0.5,4,TRUE,TRUE,0.8,"green,red","normal","Temp,Press","split")

sensorNames: Range with sensor names (A1:C1 for multiple sensors)

startDatetime, endDatetime: Time range

Optional: points(100), height(0.3), aspectRatio(5), showLine(true), showPoints(true)

Optional: opacity(0.9), colors("green,red,blue"), style("sparkline")

Optional: labels("Sensor 1,Sensor 2"), yAxisBehavior("share" or "split")

Multi-sensor: Use comma-separated colors/labels, "split" for dual y-axis

Returns: Complete Python matplotlib code with enhanced multi-sensor support

3 Tips

  • • Timestamp columns return Excel serial numbers - format as Date to see readable times
  • • Use ranges (A1:C1) for multiple sensors in PULL and CREATE_PLOT_CODE
  • • Time formats: "2024-01-01T00:00:00" or "8/1/2025 0:00" both work
  • • Grid intervals: "15min", "1h", "2h", "1d", "30s" etc.
  • • Colors: "red", "blue", "green" or hex codes like "#FF0000"
  • • Multi-sensor plotting: Use comma-separated colors/labels ("red,blue", "Temp,Press")
  • • Y-axis modes: "share" (overlay sensors) or "split" (dual y-axis for 2 sensors)
  • • Plot styles: "sparkline" (minimal), "minimal" (basic axes), "normal" (full grid)
