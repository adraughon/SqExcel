Seeq Authentication

Getting Started

  1. Create Access Key: In Seeq: Profile → Access Keys → Create. Copy Key and Password.
  2. Authenticate: Enter URL, Key, Password above → Click Authenticate.
  3. Use in Excel: Type the functions below in cells.

Quick Functions

Copy/paste these into Excel:

=SEARCH_SENSORS(A1:C1)
=PULL(A1:C1,"2024-01-01T00:00","2024-01-31T23:59")
=PULL(A1:C1,"2024-01-01T00:00","2024-01-31T23:59","grid","15min")
=CURRENT("Area/Tag")
=AVERAGE("Area/Tag","2024-01-01T00:00","2024-01-31T23:59")

Tip: Timestamp column returns Excel serials. Format as Date to see readable times.

Troubleshooting

  • #NAME? Ensure add-in is loaded.

  • No data Check sensor names and time range.
